Object 类的使用

1. Object类是什么？

Object类是所有Java类的根父类

2. 默认父类

如果在类的声明中未使用extends关键字指明其父类，则默认父类为java.lang.Object类

3. 作为参数

Object类型作为方法的参数，方法可以接收任何类作为其参数使用

4. Object 类中的主要结构

  • 属性：无
  • 构造器：Object类只声明了一个空参的构造器
  • 方法：equals() / toString() / getClass() /hashCode() / clone() / finalize()
    wait() 、 notify()、notifyAll()
